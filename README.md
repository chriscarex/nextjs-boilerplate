# NextJs Boilerplate

## Introduction

NextJs Boilerplate including sass, husky, jest, eslint and more!

## Prerequisites

- [NodeJS](https://nodejs.org/en/download/)

## Installation

`npm install`

### Build

`make build`

### Run dev server

`npm run dev` # opens on http://localhost:3000

### Testing and Linting

```shell script
  npm run test
```

```shell script
  npm run lint
```
